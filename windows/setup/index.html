Setup for Windows

This will require...

  • Windows Computer
    • Chrome Remote Desktop Remote Access
  • Other Computer (Like a Chromebook)
    • Chrome Remote Desktop Site
  • Designated Account for it*
  • Some settings changes*

*Not required but will help.

Section 1: Windows Side

Step 1: Create the account you'll use. (if you haven't already)
Step 2: Open this, and follow it for the account you'll use.

Section 2: Chromebook Side

Step 1: Add the account you'll use.


New Tab > Profile Picture > Add [Another] Account.


Step 2: Open The Chrome Remote Desktop Site
 Step 3: Tap on the computer you set as that computer.


New Tab > Profile Picture > Add [Another] Account.
